# 🌌 AetherNet v1.4

AetherNet is a custom-built, multi-tabbed development browser designed to bridge the gap between local development and the live web. It allows developers to "deploy" local HTML projects to custom virtual domains (like `example.com`) for a more authentic testing experience.

## ✨ Features
*   **Virtual Domains:** Map local file paths to custom `.com` addresses.
*   **Multi-Tabbed Interface:** Just like Chrome, open multiple tabs for research and development.
*   **Theme Switcher:** Toggle between **Aether Dark** (Monokai) and **Aether Light** (GitHub) styles.
*   **OneDrive Support:** Handles complex file paths, including spaces and special characters.
*   **Path Masking:** Keeps your URL bar clean by showing your domain instead of `file:///` strings.
